# Vocabulary Upgrades — Band 6 → Band 7

> Band 7 is **not** about rare words; it is about **precise** words and **natural collocations**. The examiner's mental test is: *Does this word fit the slot, or has the writer reached for a thesaurus?* The tables below give safe, idiomatic upgrades that move writing from "adequate" (Band 6) to "flexible and precise" (Band 7).

---

## 1. The "stop using these" list (chronic Band 6 words)

| Overused word | Why it caps you at 6 | Strong replacements (pick by context) |
|---|---|---|
| *very* | Filler intensifier, low precision | *extremely, particularly, markedly, considerably, significantly*; better — drop the intensifier and use a stronger adjective: *very important → crucial* |
| *good* | Vague evaluation | *beneficial, advantageous, favourable, effective, sound, valuable* |
| *bad* | Vague evaluation | *adverse, detrimental, harmful, damaging, undesirable, problematic* |
| *big* | Imprecise size | *substantial, significant, considerable, sizeable, major, vast* |
| *small* | Imprecise size | *minor, marginal, negligible, modest, slight* |
| *important* | Overused | *crucial, vital, pivotal, essential, paramount, fundamental, critical, indispensable* |
| *thing(s)* | Vague filler | *factor, aspect, issue, element, matter, consideration* |
| *people* | Generic | *individuals, citizens, residents, members of the public, the general population, society* |
| *a lot of / lots of* | Informal | *a substantial amount of, a considerable number of, numerous, a wide range of* |
| *many* (overused) | Repetitive | *a substantial proportion of, a great many, countless, the majority of, the bulk of* |
| *get* | Slangy / vague | *obtain, acquire, receive, become, gain, secure, attain* |
| *help* | Generic | *facilitate, enable, foster, support, assist, contribute to, enhance* |
| *make* (cause) | Vague | *cause, produce, generate, bring about, give rise to, lead to, trigger* |
| *show* | Generic | *demonstrate, illustrate, indicate, reveal, suggest, highlight, depict* |
| *say* | Plain | *argue, assert, claim, contend, maintain, suggest, point out* |
| *think* | Plain | *believe, consider, contend, hold the view that, take the view that* |
| *use* | Plain | *employ, utilise, apply, exploit, draw on, make use of* |
| *change* | Plain | *transform, alter, modify, shift, transition, evolve, reshape* |
| *problem* | Generic | *issue, concern, challenge, dilemma, drawback, predicament, pitfall* |
| *solve* | Plain | *address, tackle, resolve, alleviate, mitigate, combat, remedy* |

## 2. Common-phrase → upgrade table

| Band 6 phrase | Band 7 phrase |
|---|---|
| *very important* | *of paramount importance / crucial / pivotal* |
| *very common* | *widespread / pervasive / prevalent* |
| *very different* | *markedly different / fundamentally distinct* |
| *bad effect* | *adverse impact / detrimental consequence* |
| *good effect* | *beneficial impact / positive outcome* |
| *a lot of money* | *a substantial sum / considerable financial resources* |
| *a lot of people think* | *it is widely believed that / a substantial proportion holds the view that* |
| *I think* | *I would argue / it seems to me that / from my perspective* |
| *In my opinion* (overused) | *I take the view that / I am inclined to believe that* |
| *because of this* | *as a consequence / for this reason / on this account* |
| *for example* | *to illustrate / a case in point is / one clear example is* |
| *nowadays* | *in recent years / in the contemporary world / at present* |
| *in the world* | *globally / worldwide / on an international scale* |
| *it is bad* | *this is undesirable / this poses serious concerns* |
| *kids* | *children / young people / adolescents / minors* |
| *old people* | *the elderly / older adults / senior citizens* |

## 3. Hedging — the academic register marker

Band 7 writers avoid absolute claims. Hedging reveals control of register.

| Too absolute (Band 6) | Hedged (Band 7) |
|---|---|
| *Technology causes addiction.* | *Technology **can** contribute to addictive behaviour.* |
| *All teenagers waste time online.* | *Many teenagers **tend to** spend excessive time online.* |
| *This is the reason.* | *This **may be** one of the underlying reasons.* |
| *People are unhappy.* | *People **often appear** less satisfied.* |
| *Globalisation destroys culture.* | *Globalisation **arguably** erodes some cultural traditions.* |

### Hedging toolkit

| Function | Devices |
|---|---|
| Modal verbs | *may, might, could, would, can* |
| Lexical verbs | *seem, appear, tend to, suggest, indicate* |
| Adverbs of frequency | *often, sometimes, usually, frequently* |
| Modal adverbs | *probably, possibly, perhaps, arguably, presumably* |
| Quantifiers | *many, some, a number of, a proportion of, the majority of* |
| Distancing | *it is widely believed that / it has been suggested that / it could be argued that* |

## 5. Functional academic phrases (use these throughout an essay)

### 5.1 Introducing an argument

- It is widely believed that…
- One school of thought maintains that…
- Proponents of this view argue that…
- A common perception is that…

### 5.2 Adding a point

- A further consideration is…
- What is more important, however, is that…
- This is compounded by the fact that…
- Beyond this, it should be noted that…

### 5.3 Showing contrast

- That said, …
- This is not to say that…
- While it is true that…, it must also be acknowledged that…
- Conversely, …
- A counter-argument can be made that…

### 5.4 Showing cause and effect

- *stem from, give rise to, trigger, exacerbate, mitigate, culminate in, attribute X to Y, account for*
- This results in…
- One consequence of this is…
- This, in turn, leads to…

### 5.5 Giving examples

- A case in point is…
- One clear illustration is…
- This can be exemplified by…
- Take, for instance, …

### 5.6 Concluding / synthesising

- On balance, it appears that…
- Weighing both perspectives, …
- The evidence suggests that, while…, ultimately…
- In light of the arguments above, …

## 7. The Band 7 paraphrasing toolkit

Examiners specifically reward paraphrasing of the prompt without distortion of meaning.

| Prompt word | Safe paraphrases |
|---|---|
| *increase* | rise / grow / climb / surge / escalate / soar |
| *decrease* | decline / drop / fall / diminish / dwindle / contract |
| *cause* | trigger / give rise to / lead to / bring about / generate |
| *effect* | impact / consequence / outcome / repercussion |
| *children* | young people / minors / the younger generation / school-age children |
| *advantage* | benefit / merit / upside / positive aspect |
| *disadvantage* | drawback / downside / pitfall / negative aspect |
| *agree* | concur / share the view / endorse |
| *disagree* | dispute / contest / take issue with |
| *reason* | factor / driver / underlying cause / motivation |
| *solution* | remedy / measure / approach / strategy |
| *country* | nation / state |
| *people in the world* | the global population / individuals worldwide |
| *more and more* | a growing number of / an increasing proportion of |
| *modern* | contemporary / present-day |

**Warning:** Do not over-paraphrase to the point of distorting meaning. *"School children"* should not become *"academic pupils of juvenile age"*. If a synonym sounds odd, keep the original word.

## 8. Less-common single words that consistently impress when used correctly

(Use sparingly — one or two per paragraph, only if you are 100 % sure of meaning and collocation.)

*paramount, pivotal, indispensable, prevalent, pervasive, conducive, detrimental, sustainable, viable, plausible, inevitable, unprecedented, comprehensive, ambiguous, contentious, controversial, prudent, robust, scarce, abundant, profound, marginal, intrinsic, extrinsic, tangible, intangible, sceptical, vulnerable, susceptible*

## 9. Words to avoid (memorised cliches examiners flag)

- *Two-edged sword / double-edged sword* (overused)
- *In a nutshell*
- *Last but not least*
- *Each and every*
- *Hustle and bustle of the city*
- *Stepping stone*
- *Cherry on the cake*
- *Time waits for no one*
- *In the modern era of globalisation* (cliché opener)
- *It is a controversial topic in modern society* (cliché opener)
- *Pros and cons* (acceptable in speech, weak in writing — prefer *advantages and disadvantages*)
