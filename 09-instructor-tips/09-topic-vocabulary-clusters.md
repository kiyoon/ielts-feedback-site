## How to use this file when marking

When the candidate writes on a topic listed above:

1. **Audit the topic-specific vocabulary** — does the essay use 4–6 of the cluster words? If not, recommend specific upgrades.
2. **Check verb collocation density** — band 7+ writers use 1–2 strong topic collocations per body paragraph (`wreak havoc on`, `bridge the gap`, `bolster the immune system`).
3. **Avoid generic alternatives** — if the candidate writes "good for the environment", the topic file says "conducive to environmental sustainability" or "eco-friendly".
4. **Don't over-recommend rare phrases** — the candidate using `wreak havoc on` once is band-7-coded; using it three times is unnatural.
