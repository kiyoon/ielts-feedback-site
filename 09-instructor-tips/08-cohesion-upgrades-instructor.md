# Cohesion Upgrades — Beyond Firstly / Secondly

> **Compiled from:** Chinese (学为贵, 知乎, 51offer, 360intedu, kmf, Spiiker), Korean (해커스, scorefactory), Vietnamese (IELTS Fighter), Japanese (solo-ielts).
>
> Mechanical `Firstly / Secondly / Finally` caps Coherence & Cohesion at **band 6.5**. Drop-in upgrades + instructor-style cohesion below.

---

## 1. Why mechanical linkers cap C&C at 6.5

🇨🇳 (kmf 雅思考满分) instructor-cited band-7 descriptor:

> "Uses a range of cohesive devices appropriately, although there may be some over- or under-use."

Translation: writers stuck at 6.5 over-use exactly four words: `Firstly`, `Secondly`, `Furthermore`, `In conclusion`. The only fix is a wider, **more idiomatic** repertoire.

🇰🇷 (해커스 Jen) advice: **vary the position** of the linker. "Firstly" at the start of every paragraph is mechanical; embedding `What is more compelling` or `One factor that warrants close attention` in the middle of a sentence reads as natural cohesion.

## 2. Addition / "Furthermore" alternatives

| Common (band 6) | Upgrade (band 7+) |
|---|---|
| Also | What is more / Moreover / Furthermore / Additionally / In addition / Beyond this / On top of this |
| And | along with / coupled with / together with / not to mention / as well as |
| Another | A further / An additional / A second / Yet another / One more |
| Another reason | A further reason / A more compelling argument / Building on this point / Adding to this |

### Drop-in idiomatic phrases (instructor-style)

```
- "Building on this point, …"
- "Adding weight to this argument is …"
- "A more compelling case can be made, however, that …"
- "What is more telling is that …"
- "What is more concerning is that …"
- "Compounding this issue is …"
- "Going further still, …"
- "Beyond the immediate effect, …"
```

## 3. Contrast / "However" alternatives

| Common | Upgrade |
|---|---|
| But | However / Nevertheless / Nonetheless / Yet / Even so / That said |
| On the other hand | By contrast / Conversely / In contradistinction / On the contrary |
| Although | Despite the fact that / In spite of / Notwithstanding / Even though / While / Whereas |
| But also | But equally / But at the same time / But by the same token |
| In contrast | By way of contrast / In opposition / Conversely / On the flip side (informal — use sparingly) |

### Drop-in idiomatic phrases

```
- "That said, …"
- "Be that as it may, …"
- "This is not to say, however, that …"
- "While this view has merit, it overlooks …"
- "Critics of this position would argue that …"
- "It is here that the argument begins to falter."
- "A counter-argument worth weighing is that …"
```

## 4. Cause / "Because" alternatives

| Common | Upgrade |
|---|---|
| Because | As / Since / Owing to the fact that / On the grounds that / Given that |
| Because of | Owing to / Due to / On account of / As a result of / Stemming from / Attributable to |
| The reason is | The rationale lies in / The underlying cause is / What underpins this is |
| This is because | This stems from / This can be traced to / This finds its roots in |

🇰🇷 (Korean cram-school): never use `because` more than once per paragraph. Mix in `as / since / given that`.

## 5. Effect / "So" alternatives

| Common | Upgrade |
|---|---|
| So | Therefore / Consequently / Hence / Thus / Accordingly / As a result |
| So that | In order that / With the result that |
| As a result | Consequently / The upshot is that / The corollary is that |
| Therefore | Accordingly / It follows that / On these grounds / Hence / Thus |
| For this reason | In light of this / Given this / On these grounds / In view of this |

### Drop-in
```
- "It thus comes as no surprise that …"
- "This, in turn, leads to …"
- "The knock-on effect is …"
- "The ramifications of this are far-reaching: …"
- "By extension, …"
- "What follows is that …"
```

## 6. Example / "For example" alternatives

| Common | Upgrade |
|---|---|
| For example | For instance / By way of example / As an illustration / To illustrate / A case in point is |
| Such as | including / namely / particularly · for one |
| Like | such as / similar to · in much the same way as |
| Take X | Take, for instance, X / Consider X / Look no further than X |

### Drop-in
```
- "A clear illustration of this is …"
- "This is exemplified by …"
- "A telling example is provided by …"
- "Take, for instance, the case of …"
- "One need only look at … to see this in practice."
- "The case of [X] is particularly instructive in this regard."
- "Consider, by way of example, …"
```

## 7. Sequence / order alternatives

| Common | Upgrade |
|---|---|
| Firstly / First of all | The first point worth making is that / To begin with, / Initially, / At the outset, |
| Secondly | A further consideration is / The second point concerns / Adding to this, / Equally significant is |
| Thirdly | A third factor is / Yet another consideration is / Beyond these, |
| Finally | Finally, / Lastly, / The final point worth making is that / Ultimately, |
| Last but not least (⚠ avoid) | The most compelling point, perhaps, is that / Equally important — and often overlooked — is |

🇨🇳 (51offer / 360intedu) instructor warning: do not use a sequencer in **every** paragraph. Replace half of them with content-based linkers ("Building on the previous point, …", "A related concern is …").

## 8. Summary / "In conclusion" alternatives

| Common | Upgrade |
|---|---|
| In conclusion | On balance / All things considered / Weighing up the evidence / To draw the threads together / In summary / On reflection / Taking everything into consideration |
| To sum up | To recap / To recapitulate / Bringing the various strands together |
| All in all | On the whole / By and large / Overall / Broadly speaking |

## 10. Concession / "Although"

| Common | Upgrade |
|---|---|
| Although | While / Whereas / Even though / Notwithstanding / In spite of / Despite |
| In spite of | Notwithstanding / Despite / Regardless of |
| Even if | Even supposing / Even granting that / Granting that |
| It is true that | Admittedly, / Granted, / There is no denying that |

### Drop-in
```
- "Admittedly, [counter-position]. However, [pivot]."
- "Granted, [counter-position]. Nevertheless, [main position]."
- "While there is some truth in the claim that …, this overlooks …"
- "It is true that … But this does not negate the fact that …"
```

## 11. Lexical cohesion — the **band-7 backbone**

🇨🇳 (kmf): 7+ writers achieve cohesion not through linkers alone but through:

1. **Synonym chains** — "the government → the authorities → policymakers → the state → the administration"
2. **Pronoun reference** — using "this", "such", "the latter", "the former" precisely
3. **Repeated key concepts under different forms** — "education" / "schooling" / "academic instruction"
4. **Demonstrative this/that** — "**This view** overlooks the fact that …" (refers back to the previous sentence)

### Demonstrative-summary phrases (drop-in)

```
- "This view overlooks the fact that …"
- "Such an approach risks …"
- "The latter argument carries greater weight because …"
- "The former concern, while valid, can be addressed by …"
- "This phenomenon is most pronounced in …"
- "Such trends are not without consequence: …"
- "These developments collectively suggest that …"
```

## 12. Pronoun reference — common errors

⚠ 🇨🇳 / 🇰🇷-flagged errors (from 知乎 / 해커스):

| Wrong | Why | Right |
|---|---|---|
| "It is important." (no clear antecedent) | "It" is dummy; needs setup | "This issue is important." / "It is important that students learn …" |
| "They say that …" (vague "they") | Who is "they"? | "Critics argue that …" / "A common view holds that …" |
| Excessive "this" / "that" | Overuse loses precision | Vary with "such an approach", "the former", "the issue at hand" |
| "The government should solve this. They have the resources." | "the government" is singular | "It has the resources." (or use "policymakers" → plural) |

## 13. Paragraph-to-paragraph cohesion

🇰🇷 (해커스 Jen) advice: each paragraph should **echo a key word** from the previous paragraph in its first sentence. This creates flow without mechanical linkers.

```
[End of Body 1]    "… and the long-term implications for public health are equally serious."
[Start of Body 2]  "Beyond public health, however, lies an even more pressing concern: …"
```

| Paragraph link types | Drop-in opener |
|---|---|
| Stronger reason | "A more compelling argument, however, is that …" |
| Counter-balance | "On the other side of the debate, …" |
| Build / extend | "Building on this point, …" |
| Pivot | "While the above considerations are valid, …" |
| Specify | "Of particular significance here is …" |
| Generalise | "More broadly, …" |
| Zoom in | "Looking more closely at this issue, …" |
| Zoom out | "Stepping back from the immediate question, …" |

## 14. The instructor-style cohesion phrases (curated)

## 15. ⚠ Linkers to AVOID (cross-source consensus)

| Avoid | Why |
|---|---|
| "Last but not least" | cliché, signals memorised template |
| "All in all" | too informal for academic writing |
| "First, second, third" (numerical) | mechanical; replace at least two of them |
| "What's more" (contracted) | informal contraction; use "what is more" |
| "Plus" / "Besides" (sentence-initial) | informal |
| "By the way" | conversational |
| "Anyway" | conversational |
| "In a nutshell" | informal |

---

## 16. Cohesion audit for markers

When marking, score cohesion by counting:

1. Total linkers used → if **>1 per sentence on average**, the writing is over-cohesived (caps C&C at 6.5).
2. Distinct linkers → if fewer than **6 distinct linkers across the essay**, suggest variety.
3. Mechanical sequencers (`Firstly / Secondly / Finally`) → if **all three are used**, replace at least one with a content-based linker.
4. Demonstrative `this / that` → check each one has a clear antecedent. If "it is important" or "this is true" appears without a clear antecedent, flag.
5. Paragraph openings — does each new paragraph echo a word from the previous one? If not, suggest a connecting phrase.
