## Cross-reference: which topics tend to appear in which question type

IELTS Task 2 has five canonical question types. Some topics map disproportionately to particular types — preparing on the right type matters.

| Question type | Definition (in plain terms) | Topics that lean here |
|---|---|---|
| **Opinion (agree/disagree, "to what extent")** | Take a clear position on one claim | `01-education`, `04-health`, `06-government`, `15-sport`, `17-science` |
| **Discuss both views + your opinion** | Lay out two sides, then commit | `05-society`, `08-globalisation`, `09-media`, `11-family`, `12-crime`, `16-arts` |
| **Advantages / disadvantages (sometimes "outweigh")** | Weigh upsides vs downsides | `03-technology`, `07-urbanisation`, `10-work`, `14-tourism` |
| **Problem / solution (or causes / solutions)** | Diagnose a problem, propose remedies | `02-environment`, `04-health` (lifestyle), `07-urbanisation` (traffic/housing), `09-media` (fake news), `12-crime` |
| **Two-part question** | Two related sub-questions inside one prompt | Any topic — most common in `01-education`, `02-environment`, `06-government`, `13-money` |

A topic-bank file's `Ready-to-adapt thesis statements` section gives one thesis variant for each of the five question types so the student can match the prompt format quickly.

## How topics overlap (very common in real exams)

Real prompts usually combine **two or three** thematic threads. The agent should consult **all** matching files. Common pairings:

| Health + government | "Should governments tax sugary drinks?" | 04 + 06 + 13 |

## Recommended preparation sequence (for a band-6 student aiming at band 7)

If a student has limited prep time, prioritise in this order:

1. **Tier 1** (`01, 02, 03, 04, 06`) — five files = ~80 % of likely prompts.
2. Then add **Tier 2** topics that match the student's life experience (work topic for working professionals; family topic for parents; etc.).
3. Skim Tier 3 / 4 last; their value is the **vocabulary clusters**, less the arguments.

For each prioritised file, the student should:
1. Read the `Sub-topics` and `Sample prompts` sections to recognise the prompt family.
2. Memorise **8-10 items** from the C1 vocabulary cluster (not all 25+).
3. Memorise **2 thesis statement templates** (their two most-likely question types).
4. Read the two sample paragraphs **once a week** for rhythm — then practise writing their own on the same prompt without looking.
