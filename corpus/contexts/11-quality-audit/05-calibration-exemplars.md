# 05 — Calibration Exemplars

These are the **best-calibrated** samples per band, per task — the samples where descriptor evidence most strongly supports the claimed band. An LLM marker can use these as anchors when applying the descriptors to fresh student writing.

> **Reading rule:** Anchor only on samples in this list. Do **not** anchor on samples flagged in `06-flagged-samples.md`. For samples not on either list, treat the claimed band as a soft guide.

## How to use these exemplars

For an LLM marker:
1. When a fresh student essay shows similar TR/CC/LR/GRA features to one of these anchors, you can confidently apply the same band.
2. For features in between two anchors (e.g. between Band 6 Sample 1 and Band 7 Sample 6), apply 6.5.
3. Do **not** anchor on samples in `06-flagged-samples.md` — the descriptor evidence does not support the claimed band.
4. Prefer anchors from **British Council, IELTS Liz, IELTS Advantage, ieltspodcast** (Tier 1) over engnovate/writing9 anchors when both exist for the same band.
