# 03 — Task 1 Academic Sample Audit

This document audits every Task 1 Academic sample in `contexts/02-task1-academic/` against the official Task 1 Academic descriptors. The Task 1 sample files contain a mix of (a) reproduced answers, (b) summarised answers + examiner notes, and (c) **composite / typical-pattern descriptions** which are not real candidate responses. The audit notes which is which.

**Inflation pattern:** Writing9's per-criterion sub-scores are systematically broken (TA=9 + CC=9 + LR=4 + GRA=4). The overall labels usually still land near the right band (6.0–6.5), but the per-criterion mix reported in the source files cannot be used to teach the descriptors.

**Composite warning:** ~20% of "samples" in the Task 1 Academic file set are composite typical-patterns, not real candidate writing. The Band 5.5 file in particular is half-composite. Users should not anchor calibration on these.

**Model-vs-candidate confusion:** The Band 7.0 file mixes real Band 7 candidate writing with Band 9 models. The Band 9 models are correctly labelled as such within the entries, but a reader skimming the file might mistake them for Band 7 candidate output. This is a **mislabelling** at the file level, even though the inline labels are accurate.
