## 1. "Important" / "very important"

🇨🇳 (xdf 新东方 + zim.vn): the single most-overused word at band 6 is `important`. There are 12+ band-7 alternatives.

| Common (band 6) | Upgrade (band 7+) |
|---|---|
| important | crucial / vital / essential / pivotal / paramount / indispensable / significant / fundamental / cardinal / instrumental |
| very important | of paramount importance / of utmost significance / critically important |
| more important than | takes precedence over / outweighs / supersedes |
| It is important to | It is imperative to / It is essential to / It is vital that we |
| an important role | a pivotal role / a fundamental role / a decisive role / an instrumental role |
| play an important role in | be instrumental in / be pivotal to / play a key role in |
| not important | trivial / negligible / insignificant / marginal / inconsequential |

```
- Cultivating critical thinking is **of paramount importance** in higher education.
- The role of family **is instrumental in** shaping a child's worldview.
- Government investment in renewable energy **takes precedence over** short-term subsidies.
```

## 2. "Good" / "bad"

🇻🇳 (zim.vn) recommended the 7 band-7 replacements for `good`: **decent, satisfactory, acceptable, passable, competent, fine, qualified**.

| Common | Upgrade |
|---|---|
| good (general) | beneficial / advantageous / favourable / positive / constructive / sound |
| good (skilled) | competent / proficient / adept / accomplished / qualified |
| good (acceptable) | satisfactory / adequate / acceptable / passable / decent |
| good (high quality) | excellent / outstanding / exceptional / first-rate / superb |
| a good thing | a positive development / a constructive step / a favourable outcome |
| It is good for | It is beneficial to / It does X good / It contributes positively to |
| bad (general) | detrimental / harmful / damaging / adverse / unfavourable / negative |
| bad (severe) | dire / disastrous / devastating / catastrophic |
| bad (immoral) | unethical / immoral / reprehensible / unacceptable |
| bad effects | adverse effects / detrimental impacts / negative consequences / damaging repercussions |

```
- Excessive screen time has **detrimental impacts** on children's cognitive development.
- Public transport offers a **constructive step** towards reducing emissions.
- Long working hours can be **disastrous** for family relationships.
```

## 4. "Increase / decrease" — Task 2 abstract usage

|---|---|

| increase confidence | boost / instil / foster confidence |
| increase pollution | exacerbate / aggravate / intensify pollution |
| decrease pollution | mitigate / curb / alleviate / reduce pollution |
| decrease crime | curb / clamp down on / suppress crime |
| decrease unemployment | tackle / address / reduce / combat unemployment |


## 5. "Problem / solution"

🇰🇷 (해커스 고숭이): "problem-solution essays should not contain the word 'problem' more than twice. Replace with `issue / challenge / dilemma`."

| Common | Upgrade |
|---|---|
| problem | issue / challenge / dilemma / predicament / concern / setback |
| serious problem | pressing issue / acute challenge / dire predicament |
| cause a problem | give rise to / engender / spark / trigger / precipitate |
| solve a problem | tackle / address / resolve / mitigate / alleviate / remedy |
| solution | remedy / measure / countermeasure / approach / response |
| effective solution | viable / workable / sustainable / pragmatic solution |
| temporary solution | a stopgap / a quick fix / a short-term remedy |
| long-term solution | an enduring / lasting / sustainable approach |

🇯🇵 (solo-ielts-toefl) drop-in cause-and-solution phrases:

```
- "One possible quick fix could be…"
- "The key to unraveling the issue is…"
- "The origins of the problem can be traced to…"
- "The genesis of this issue lies in…"
- "A more sustainable approach would be to…"
```

## 8. "Agree / disagree" — opinion-stating frames

🇰🇷 (해커스 고숭이) and 🇨🇳 (老烤鸭): never write `I think` or `I agree`. Upgrade obligatory.

| Common | Upgrade |
|---|---|
| I agree | I am inclined to agree / I subscribe to the view that / I am of the opinion that |
| I strongly agree | I firmly believe that / I wholeheartedly endorse the view that |
| I disagree | I am of a contrary opinion / I beg to differ / I take issue with |
| I think | It is my contention that / In my considered view / I would argue that |
| Some people think | A common perception is that / It is widely held that / Many would argue that |
| In my opinion | From my perspective / In my considered judgement / To my mind |

🇯🇵 (solo-ielts-toefl) opinion drop-ins:

```
- "My frank and honest opinion is that…"
- "It is my firm belief that…"
- "I am inclined to believe that…"
- "From where I stand,…"
- "Personally, I subscribe to the view that…"
```
